## Core Panel Tabs

<Tabs>
<TabItem value="console" label="Console" default>

### Purpose
Real-time server monitoring and interaction

<Collapse title="Key Features">
- **Live Logs**  
  ```log
  [15:32] User 'Player123' joined the game
  [15:33] WARN: Low memory warning
  ```
- **Command Execution**  
  ```bash
  # Minecraft example
  give Player123 minecraft:diamond 64
  ```
- **Emergency Controls**  
  <div className="button-group">
    <button onClick={() => console.log('Restarting...')}>🔄 Soft Restart</button>
    <button onClick={() => console.log('Killing...')}>⛔ Force Stop</button>
  </div>
</Collapse>

</TabItem>

<TabItem value="files" label="Files">

### Purpose
File management and configuration

<Collapse title="Key Features">
```file-tree
/server
  ├── config/
  │   └── server.properties
  ├── mods/
  │   └── BiomesOPlenty.jar
  └── backups/
```

**File Actions**  
```javascript
// Example file permission
chmod 755 /server/config/server.properties
```

<div className="grid">
  <button onClick={() => console.log('Uploading...')}>📤 Upload Files</button>
  <button onClick={() => console.log('Zipping...')}>🗜️ Compress Folder</button>
</div>
</Collapse>

</TabItem>

**Security Settings**

</TabItem>
