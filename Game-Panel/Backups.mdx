## **2. How to Create a Backup**

### **Manual Backup**

1. Click **"Create Backup"** (top-right).
2. Optional: Add a **description** (e.g., "Pre-Update Backup").
3. Click **"Start Backup"** ‚Üí Wait for completion (progress bar appears).

‚û°Ô∏è **___Backups include all files, databases, and configurations.___**

### **Automated Backups**

1. Go to **Schedules** tab ‚Üí Create a new schedule.
2. Set:
   - **Action:** `Create Backup`
   - **Frequency:** (e.g., `0 3 * * *` for daily 3 AM backups)
   - **Retention:** Keep last `X` backups (prevents disk overload).

## **3. Managing Backups**

### **Backup Actions (‚ãÆ Menu)**

| **Action**   | **Purpose**                                                    |
| :----------- | :------------------------------------------------------------- |
| **Download** | Save backup locally (`.zip`/`.tar.gz`)                         |
| **Restore**  | Roll back server to this backup _(overwrites current files\!)_ |
| **Lock**     | Prevent auto-deletion (for critical backups)                   |
| **Delete**   | Free up disk space                                             |

### **Restoring a Backup**

1. Click **‚ãÆ ‚Üí Restore** on the desired backup.
2. Confirm ‚Üí Server will **stop**, restore, then **restart**.
3. Check **Console** for errors.

‚ö†Ô∏è **Warning**:

- Restoring **overwrites all current files**.
- Ensure the server is **backed up again** before restoring.

---

## **4. Backup Settings**

### **Retention Rules**

- Set in **Schedules** ‚Üí `Create Backup` action.
- Example: Keep last `7` backups ‚Üí Older ones auto-delete.

### **File Inclusions/Exclusions**

- By default, backups include:
  - All server files
  - Databases
  - Configs
- Exclude files by adding to `.pterodactyl-backup-ignore` (advanced).

---

## **5. Troubleshooting**

| **Issue**     | **Solution**                                      |
| :------------ | :------------------------------------------------ |
| Backup fails  | Check **disk space** (7.96 GiB free in your case) |
| Restore stuck | Verify server **stopped** properly first          |
| Slow backups  | Reduce backup size (exclude logs/worlds)          |
