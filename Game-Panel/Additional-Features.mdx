## **Overview**

This guide covers all aspects of managing plugins and modpacks through your XGamingServer panel. Both tabs share similar filtering controls but serve different purposes for enhancing your server.

## **Shared Interface Controls**

### **Provider Selection**

Choose where to get your content from:

- **Modrinth**: Best for modern, optimized content (supports both plugins and modpacks)
- **SpigotMC**: Traditional plugin source (plugins only)
- **CurseForge**: Official modpacks with automatic updates (modpacks only)
- **FTB**: Popular modpack collections (modpacks only)
- **Custom**: Upload your own files (both)

### **Page Size Options**

Control how many items display:

- **10 items**: Best for mobile viewing
- **25 items**: Default balanced view
- **50 items**: For power users needing to compare many options

### **Search Features**

Find exactly what you need with:

- Name searches (`Essentials`)
- Author filters (`author:lucko`)
- Version matching (`version:1.20.1`)

## **Plugins Tab Details**

![X Gaming Server Minecraft Plugins Screenshot Sv](/images/XGamingServer_MinecraftPluginsScreenshot.svg)

### **Plugin Loader Compatibility**

Different server types require specific plugin formats:

- **Paper**: Standard for most servers (1.8\+)
- **Folia**: For advanced multi-threaded servers (1.19.4\+)
- **Spigot**: Legacy format for older servers

### **Installation Process**

1. **Find plugins** using filters
2. **Check compatibility** with your server type
3. **Install with dependencies** (auto-resolved)
4. **Configure settings** through the built-in editor

### **Management Tools**

- **Live reloading**: Apply changes without restart
- **Version control**: Rollback to stable versions
- **Dependency viewer**: See all connected plugins

## **Modpacks Tab Details**

### **Modpack Types**

- **Lite packs**: Small performance-focused collections
- **Complete packs**: Full gameplay experiences
- **Mega packs**: Hundreds of mods for advanced users

### **Installation Process**

1. **Check requirements** (RAM, storage)
2. **Select components** (optional mods)
3. **Automatic setup** of configs and permissions
4. **Optimize settings** for your hardware

### **Special Features**

- **Profile system**: Save different configurations
- **Mod tweaker**: Customize individual mod settings
- **Update channels**: Choose stable or beta versions

## **Best Practices**

### **For Plugins**

- Test new plugins on a backup server first
- Keep dependencies updated
- Use the config editor for safe changes
- Monitor performance after installing

### **For Modpacks**

- Verify your server has enough resources
- Check mod compatibility lists
- Start with recommended settings
- Back up before major updates

## **Troubleshooting Help**

### **Common Plugin Issues**

- Not loading? Check server type matches plugin format
- Missing features? Verify all dependencies installed
- Causing lag? Check for resource-heavy plugins

### **Common Modpack Issues**

- Crash on start? Verify RAM allocation
- Missing mods? Check the installation log
- Performance problems? Try optimized versions
