## Getting Started

<Note>
  Always keep your API keys secure. Never commit them to public repositories.
</Note>

## Authentication Methods

<Steps>
  <Step title="Step 1">
    Obtain your API key from the dashboard
  </Step>
  <Step title="Step 2">
    Include it in request headers
  </Step>
  <Step title="Step 3">
    Verify your domain
  </Step>
</Steps>

## Code Examples

<CodeGroup>
  <CodeBlock title="JavaScript">

const response = await fetch('https://api.example.com', {

    'Authorization': `Bearer ${API_KEY}`
