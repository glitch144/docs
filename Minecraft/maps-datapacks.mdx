## Overview  

- **Custom Maps**: Replace the default world with a pre-built map.  
- **Datapacks**: Lightweight alternatives to mods that add new features without additional software.  

---

## Steps to Add Custom Maps  

### Step 1: Upload a Custom World  
1. Use the **File Manager** to locate the `world` folder.  
2. Replace the `world` folder with your custom map.  

### Step 2: Restart the Server  
- Restart the server to load the new world.  

---

## Steps to Add Datapacks  

### Step 1: Locate the Datapack Folder  
1. Navigate to `world/datapacks` via the **File Manager**.  

### Step 2: Upload Datapacks  
1. Upload `.zip` or `.jar` files for your datapacks.  

### Step 3: Restart the Server  
- Restart the server to load the datapacks.
